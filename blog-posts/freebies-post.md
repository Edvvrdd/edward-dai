## Python Scripts
Automation tools and utilities for audio workflows that can save hours of manual work.

### Batch Audio Converter
Converts multiple audio files between formats (WAV, MP3, OGG, FLAC) with customizable sample rate and bit depth. Perfect for preparing assets for different game engines.

**Features:**
- Batch processing
- Format conversion
- Sample rate control

### FMOD Event Exporter
Exports FMOD event data to CSV for documentation and analysis. Includes parameter ranges, event paths, and asset references.

**Features:**
- CSV export
- Event documentation
- Parameter tracking

### Audio File Organizer
Automatically organizes audio files into folders by type, sample rate, or custom metadata. Saves hours of manual sorting.

**Features:**
- Auto-categorization
- Metadata reading
- Custom rules

## Phase Plant Patches
Custom presets for sound design and game audio that are ready to use in your projects.

### Cinematic Whoosh Pack
10 customizable whoosh presets perfect for UI transitions, scene changes, and cinematic moments. Includes macro controls for speed and intensity.

**Features:**
- 10 presets
- Macro controls
- Game-ready

### Ambient Texture Generator
Evolving ambient textures with randomized modulation. Great for creating unique atmospheric layers that never repeat exactly.

**Features:**
- 8 presets
- Randomization
- Evolving textures

### Impact Designer
Layered impact sounds with customizable weight, material, and resonance. Perfect for footsteps, hits, and environmental interactions.

**Features:**
- 5 presets
- Material layers
- Resonance control

## FMOD Blueprints
Reusable FMOD Studio project templates and event structures that demonstrate best practices for interactive audio implementation.

### Footstep System Template
Complete footstep system with material switching, surface detection, and variable speed/weight parameters. Includes example events and parameter setup.

**Features:**
- Material switching
- Speed variation
- Weight parameters

### Adaptive Music Template
Horizontal and vertical re-orchestration template with transition markers and intensity parameters. Ready to drop in your own music stems.

**Features:**
- Stem-based mixing
- Transition markers
- Intensity control
